Projects:

All applications/projects, that I have developed or worked on, make use of open source libraries extensively. So, following are some of the libraries I’m contributing back to the community.

Pustike Inject

Implementation of JSR-330: Dependency Injection for Java.

Pustike EventBus

Event Bus based on publish-subscribe pattern.

Pustike Persist

Object based Persistence API with features like, annotation based entity mapping, schema generation, transaction management and fluent query/data finder api.

Pustike JSON

Provides Object Mapper that can create Java objects from JSON or create JSON value from Java objects, using Jakarta JSON-P APIs.

Pustike Web

RESTful Web services using JAX-RS like API.

Pustike Maven Parent

Contains common Maven configuration for Pustike projects.
